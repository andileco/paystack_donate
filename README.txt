INTRODUCTION
------------

A very simple module to accept donation using the Paystack Payment gateway.
Users fill in their email, amount, and that's all. This uses the Paystack inline API
and donation is made on a single page without any redirect

 * For a full description of the module visit:
   https://www.drupal.org/project/paystack_donate

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/paystack_donate


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install the Paystack Donate module as you would normally install a
   contributed Drupal module. Visit https://www.drupal.org/docs/7/extend/installing-modules for
   further information.


CONFIGURATION
-------------

 * Just install Paystack donate and head on to the config page at
   Home » Administration » Configuration » User interface.
   (admin/config/user-interface/paystack-donate)

 * Fill all the required fields as per your requirements.

 * Click Save Configuration.

	Note: The donate link is also configurable. Do not forget to flush your cache if
	  		you change the link.
